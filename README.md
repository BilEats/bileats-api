# bileats-api


## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BilEats/bileats-api.git
   cd bileats-api
   ```

2. Initialize the project:
   ```bash
   npm init -y
   ```

3. Install dependencies:
   ```bash
   npm install express axios cheerio
   ```


## Usage

1. Start the server:
   ```bash
   node index.js
   ```
   The server will start at `http://localhost:3000`.
